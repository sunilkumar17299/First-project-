# 🎬 Movie Recommendation System

A Movie Recommendation System built using **K-Nearest Neighbors (KNN)** and **Collaborative Filtering**. This project recommends movies similar to a selected movie based on user ratings.

---

## 📌 Features

- Recommend similar movies based on user preferences.
- Uses K-Nearest Neighbors (KNN) algorithm.
- Collaborative Filtering approach.
- Fast similarity search using cosine distance.
- Simple and easy-to-use interface.

---

- Python
- Pandas
- NumPy
- Scikit-learn
- Pickle
